## 4. 治理系统导入建议

建议按“多源资产”导入，不要把 HIS 源端直接混入 8.216 ODS 主资产：

| 治理对象 | 来源字段 |
|---|---|
| 数据源 | `source_db`，固定为 `10.10.10.15:1521/his`，凭据只存 `credential_ref` |
| 系统 | 固定 `HIS_SOURCE`，显示名“东软 HIS 源端” |
| namespace/schema | `source_owner` |
| 表资产 | `full_name/source_owner/table_name/domain/table_role/include_status` |
| 字段资产 | `source_owner/table_name/column_name/data_type/nullable/comment/column_theme/is_primary_key` |
| 关系资产 | `from_table/to_table/join_keys/validation_status/validation_source` |
| ODS 覆盖映射 | `ods_same_name_covered/ods_field_coverage_ratio` |

状态映射建议：

| 资产包状态 | 治理系统状态 |
|---|---|
| `included` | `active` |
| `candidate` | `pending_review` |
| `excluded` | `excluded`，保留原因 |
| `full_pass/verified/sample_pass` | 可导入关系，置信度按验证级别区分 |
| `candidate/partial/pass_with_subset/full_pass_with_note` | 导入复核表，不直接作为强关系 |
